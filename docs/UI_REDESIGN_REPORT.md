# UI/UX 重构报告

## 范围

本轮在现有 Vue 3、Element Plus、Drogon 和 MySQL 架构上完成产品级视觉重构，没有更换技术栈，也没有用静态 Mock 替代真实接口。

重构页面：

- 登录与注册
- 学生首页
- 课程列表与课程详情
- 章节详情与重点知识
- 题库列表与题目详情
- 学习记录与错题本
- 往年试题与学生试卷上传
- 管理后台首页
- 课程、章节、知识点、题目、试卷和用户管理

## 主要改动

- 建立独立设计令牌和 Element Plus 全局覆盖。
- 重构学生端侧栏、顶部搜索、用户菜单、移动抽屉和底部导航。
- 重构管理端浅色布局，统一管理页头、筛选、表格和弹窗。
- 首页聚合现有学习记录、错题和课程数据，不显示虚假指标。
- 管理首页新增受管理员鉴权保护的 `/api/admin/stats`。
- 补齐真实章节详情接口 `/api/chapters/{chapterId}`，移除章节占位数据。
- 题目详情保留答案按需加载、学习状态、错题本、AI 缓存讲解和继续追问。

## 安全修复

- 登录页和 README 不再公开固定测试账号密码。
- MySQL 客户端不再使用硬编码默认密码。
- 生产默认日志级别由 `DEBUG` 调整为 `INFO`。
- 数据库题干、选项、答案、解析和 AI 内容统一通过安全数学组件渲染。
- `v-html` 只保留在 `FormulaContent`，输入先转义，KaTeX `trust: false`。
- 学生题目详情继续不返回标准答案、普通解析和选项正确性；答案接口按点击调用。
- 现有 JWT 生产启动校验、管理员鉴权、限流、CSP 和密钥环境变量机制保持不变。

## 工程基线

- 新增 ESLint flat config 和 `npm run lint`。
- 新增数学内容 XSS 单测。
- 新增 Playwright 桌面与手机端冒烟测试。
- CI 增加前端 lint 门禁。
- 截图脚本会检查页面异常和 HTTP 4xx/5xx 响应。

## 已知限制

- Element Plus 和全量图标注册使主包仍有约 1.19 MB，Vite 会给出体积警告；后续可按路由拆分图标与组件注册。
- 上传文件仍依赖服务器本地目录，Render 免费实例重启后无法保证持久化，生产应迁移到对象存储。
- PDF、DOC、DOCX 正文提取与 OCR 仍需独立文档解析服务。
- 本地验收期间 Aiven 公网端口不可达，因此使用 D 盘隔离 MySQL 实例导入真实 schema 完成测试；这不影响代码构建结果，但生产部署前仍需确认数据库服务状态。
